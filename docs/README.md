## 📊 Documentation Standards

All documents follow these guidelines:
- **Format:** Markdown (.md) for text, LaTeX for technical papers
- **Version Control:** All changes tracked via Git
- **Naming:** Descriptive names with dates (e.g., `design_federated_protocol_2026-01.md`)
- **Structure:** Clear sections with table of contents for longer docs
